<h2 id = "about"><b> About The Project</b></h2>

---
![screenshot](https://github.com/Phildeoner/admin-dashboard/blob/da3df8ae38753532ca5df5860ef152a0cc3c7220/Images/Screenshot%20(17).png)<br>

<b>This is an Admin dashboard project which contain:</b>
- A search bar, a notification icon and a profile detail on the nav bar.
- A sidebar which contain links such as dashboard, home, profile, messages, history, settings, etc. 
- A section which displays projects in cards which can be viewed, stared or shared.
- An announcement section for site maintenance, etc.
- A trending section that shows the trending activities
- Contains buttons for upload, new and share<br>

<h2 id = "usage"><b>Usage</b></h2>

--- 
To access and navigate around this page after executing the steps above, follow the following steps:

1. Launch the index.html file after cloning the project to your local machine.

2. The first and topmost section is the nav bar containg the search box, notification icon and profile name and image.

3. The next sectiion contains the profile image, fullname and username to the left and the new, upload and share buttons to the right.

4. By the left the section with the blue background is the sidebar.

5. To the right of the sidebar is the project cards section.

6. To the extreme right is the announcement and trending Section.

7. The Page is responsive to fit medium, large and XL screens not fully responsive for mobile devices.
